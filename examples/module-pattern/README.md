# Module Pattern Example

This example demonstrates a small reusable pattern with `for_each`, validation, and `moved {}` refactor metadata.

Use `./scripts/Initialize-Workspace.ps1 -Path ./examples/module-pattern`.
Then run `./scripts/Validate-Terraform.ps1 -Path ./examples/module-pattern` and `./scripts/Invoke-TerraformTest.ps1 -Path ./examples/module-pattern`.
