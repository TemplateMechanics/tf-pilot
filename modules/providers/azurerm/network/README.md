# azurerm/network module

Reflection-scaffolded module contract for the azurerm network capability family.

Mapped reflection prefixes:
- Resource types: azurerm_virtual_network, azurerm_subnet, azurerm_network_security_group, azurerm_route_table
- Data source types: azurerm_virtual_network, azurerm_subnet

This scaffold is intentionally safe by default and intended for incremental expansion.
