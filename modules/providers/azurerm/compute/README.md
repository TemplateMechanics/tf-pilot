# azurerm/compute module

Reflection-scaffolded module contract for the azurerm compute capability family.

Mapped reflection prefixes:
- Resource types: azurerm_linux_virtual_machine, azurerm_windows_virtual_machine, azurerm_container_app
- Data source types: azurerm_image, azurerm_virtual_machine

This scaffold is intentionally safe by default and intended for incremental expansion.
