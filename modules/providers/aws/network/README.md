# aws/network module

Reflection-scaffolded module contract for the aws network capability family.

Mapped reflection prefixes:
- Resource types: aws_vpc, aws_subnet, aws_route, aws_route_table, aws_internet_gateway, aws_nat_gateway, aws_eip
- Data source types: aws_vpc, aws_subnets, aws_route_table

This scaffold is intentionally safe by default and intended for incremental expansion.
